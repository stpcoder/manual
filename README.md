# 한국화 손상 영역 인식 시스템

## 기술 문서

본 문서는 한국화 손상 영역 인식 시스템(Korean Painting Damage Detection System)의 공식 기술 매뉴얼입니다.

본 시스템은 한국화에 존재하는 물리적 손상 부위를 고해상도 스캔 이미지로부터 자동으로 검출하고, 복원에 필요한 보충 용지 절단 레이아웃 및 위치 가이드를 생성합니다. 포항공과대학교(POSTECH) 임지훈, 제태호가 개발하였으며, 리움미술관의 협력을 통해 실제 복원 현장의 요구사항을 반영하였습니다.

### 개발 정보

- 개발: 포항공과대학교(POSTECH) 임지훈, 제태호
- 협력: 리움미술관

### 기술 스택

- Python 3.8 이상
- PyQt6 (GUI 프레임워크)
- OpenCV 4.x (이미지 처리)
- NumPy, SciPy (수치 연산)
